# OA审批权限说明

## 一、概述

本文档详细说明OA系统中各类审批流程的权限配置规则，确保审批流程的合规性和高效性。

---

## 二、审批角色定义

| 角色名称 | 角色说明 | 权限范围 |
|---------|---------|---------|
| 申请人 | 发起审批流程的人员 | 提交申请、查看进度、撤回申请 |
| 直接上级 | 申请人的直属领导 | 一级审批、退回修改 |
| 部门负责人 | 部门主管/经理 | 二级审批、跨部门协调 |
| 分管领导 | 公司副总/总监 | 三级审批、重大事项决策 |
| 财务审批人 | 财务部门指定人员 | 费用相关审批、预算审核 |
| 人事审批人 | 人力资源部门人员 | 人事相关审批、考勤审核 |
| 总经理 | 公司最高管理者 | 最终审批、特殊事项审批 |
| 系统管理员 | OA系统管理人员 | 流程配置、权限管理、数据维护 |

---

## 三、审批流程类型及权限配置

### 3.1 请假审批

| 审批节点 | 审批人 | 审批权限 |
|---------|-------|---------|
| 一级审批 | 直接上级 | 审批3天以内请假 |
| 二级审批 | 部门负责人 | 审批3-7天请假 |
| 三级审批 | 分管领导 | 审批7天以上请假 |
| 备案 | 人事部门 | 所有请假记录备案 |

**特殊规则：**
- 病假需提供医院证明
- 年假需提前3天申请
- 紧急请假可先口头申请，24小时内补单

### 3.2 报销审批

| 审批节点 | 审批人 | 审批权限 |
|---------|-------|---------|
| 一级审批 | 直接上级 | 审核费用真实性 |
| 二级审批 | 部门负责人 | 审核部门预算 |
| 三级审批 | 财务审批人 | 审核票据合规性 |
| 四级审批 | 分管领导 | 单笔5000元以上 |
| 五级审批 | 总经理 | 单笔10000元以上 |

**特殊规则：**
- 差旅费需附出差申请单
- 招待费需附招待清单
- 发票需真实有效，抬头正确

### 3.3 采购审批

| 审批节点 | 审批人 | 审批权限 |
|---------|-------|---------|
| 申请审核 | 部门负责人 | 审核采购必要性 |
| 预算审核 | 财务审批人 | 审核预算额度 |
| 比价审核 | 采购部门 | 三家比价确认 |
| 审批决策 | 分管领导 | 5000元以下 |
| 最终审批 | 总经理 | 5000元以上 |

**特殊规则：**
- 固定资产采购需走专项流程
- 紧急采购可先执行后补单（限2000元以下）

### 3.4 出差审批

| 审批节点 | 审批人 | 审批权限 |
|---------|-------|---------|
| 一级审批 | 直接上级 | 国内3天以内 |
| 二级审批 | 部门负责人 | 国内3-7天 |
| 三级审批 | 分管领导 | 国内7天以上/国外出差 |
| 备案 | 人事部门 | 出差记录备案 |

### 3.5 用印审批

| 审批节点 | 审批人 | 审批权限 |
|---------|-------|---------|
| 申请审核 | 部门负责人 | 审核用印必要性 |
| 法务审核 | 法务人员 | 合同类文件审核 |
| 审批决策 | 分管领导 | 一般印章使用 |
| 最终审批 | 总经理 | 公章/法人章使用 |

## 四、权限管理规则

### 4.1 审批人设置

1. **固定审批人**：明确指定具体人员
2. **角色审批人**：按岗位角色自动匹配
3. **代理审批人**：审批人请假/出差时的代理人
4. **会签审批**：多人同时审批，全部通过才生效

### 4.2 审批权限控制

| 控制维度 | 说明 |
|---------|-----|
| 金额控制 | 按金额区间设置不同审批人 |
| 部门控制 | 按部门设置独立审批流程 |
| 类别控制 | 按申请类别设置专项流程 |
| 紧急程度 | 紧急申请可走快速通道 |

### 4.3 代理审批规则

1. 审批人可提前设置代理人
2. 代理人拥有同等审批权限
3. 代理期间原审批人仍可审批
4. 代理关系可设置有效期

## 五、审批操作权限

| 操作 | 申请人 | 审批人 | 管理员 |
|-----|-------|-------|-------|
| 提交申请 | ✓ | ✓ | ✓ |
| 查看进度 | ✓ | ✓ | ✓ |
| 撤回申请 | ✓（未审批时） | ✗ | ✓ |
| 同意/驳回 | ✗ | ✓ | ✗ |
| 转交他人 | ✗ | ✓ | ✗ |
| 加签审批 | ✗ | ✓ | ✗ |
| 退回修改 | ✗ | ✓ | ✓ |
| 流程催办 | ✓ | ✓ | ✓ |
| 流程终止 | ✗ | ✗ | ✓ |
| 数据导出 | ✗ | ✗ | ✓ |

## 六、特殊场景处理

### 6.1 审批人空缺

- 审批人离职/调岗时，自动转交上级
- 长期空缺需及时更新审批链

### 6.2 越级审批

- 紧急情况下可申请越级审批
- 需说明越级原因并留痕

### 6.3 批量审批

- 同类小额审批可批量处理
- 批量审批仍需逐条确认

### 6.4 移动端审批

- 支持手机APP审批
- 移动端与PC端权限一致
- 重要审批建议PC端操作

---

## 七、安全与审计

### 7.1 操作日志

- 记录所有审批操作
- 保留操作时间、IP地址
- 日志不可删除、不可修改

### 7.2 数据权限

- 普通员工只能查看自己的申请
- 审批人可查看待办和已办
- 部门负责人可查看部门数据
- 管理员可查看全部数据

### 7.3 合规要求

- 审批流程需符合公司制度
- 重要审批需留痕备查
- 定期审计审批记录
